## 新增需求决策指南

### 代码放哪层

```
请求进入
  ↓
Controller ({语言})     ← 只做参数接收 + 调用 Service，不写业务逻辑
  ↓
Service ({语言})        ← 核心业务编排，事务控制在此层
  ↓
Component ({语言})      ← 可复用业务组件（算价、外部调用封装等）
  ↓
Mapper ({语言} + XML)   ← 数据访问，复杂查询放 extend/ XML
```
